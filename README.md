# Pain Radar

45 app ideas mined from Reddit complaints, judged by Amon Studio.

Open [index.html](index.html) in a browser to browse the gallery.

Each folder contains:
- `index.html` — interactive Phase 1 prototype (8 iPhone screens)
- `spec.md` — Phase 1 spec
- `product_log.md` — origin signal, evidence, open questions
- `judgments.md` — verdicts from 3 judges

Generated by the Pain Radar pipeline. See the gallery for the full ranking.
